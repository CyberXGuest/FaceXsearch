

Installation Instructions:
For Kali Linux:

    git clone https://github.com/CyberXGuest/FaceXsearch.git
    pip install -r requirements.txt

     chmod +x install.sh && ./install.sh

     python3 face.py

  For Termux:

    Install Termux from F-Droid or Play Store

    pkg install -y git

    Clone or transfer files
    git clone https://github.com/CyberXGuest/FaceXsearch.git

    chmod +x install-termux.sh && ./install-termux.sh

    python face.py
